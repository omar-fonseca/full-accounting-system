# Full Accounting System | Gestión de Estaciones de Servicio

## 🚀 Visión del Proyecto
Sistema integral diseñado para la gestión operativa y contable de estaciones de servicio. El objetivo es centralizar el control de inventarios, la supervisión de turnos de personal y el análisis de ventas en una plataforma escalable, segura y modular.

## 🛠️ Tecnologías Actuales
- **Frontend:** React
- **Backend:** Node.js, Express
- **Base de Datos:** MongoDB Atlas
