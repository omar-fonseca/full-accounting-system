# Full Accounting System | Gestión de Estaciones de Servicio

## 🚀 Visión del Proyecto
Sistema integral diseñado para la gestión operativa y contable de estaciones de servicio. El objetivo es centralizar el control de inventarios, la supervisión de turnos de personal y el análisis de ventas en una plataforma escalable, segura y modular.

## 🛠️ Tecnologías Actuales
- **Frontend:** React
- **Backend:** Node.js, Express
- **Base de Datos:** MongoDB Atlas

## 📋 Roadmap de Desarrollo (V 1.0)
- [x] Autenticación de usuarios (Login/Register).
- [ ] Dashboard de administración (Resumen de KPIs).
- [ ] Módulo de gestión de inventarios (CRUD de aceites, filtros y repuestos).
- [ ] Gestión de turnos de personal (Control de acceso y auditoría).
- [ ] Módulo de inventario de combustible (Control volumétrico).

## 💡 Próximas Funcionalidades (Visión de Futuro)
- Análisis predictivo de demanda de combustible.
- Sistema de alertas automáticas para inventarios bajos.
- Integración con facturación electrónica.
- Reportes financieros avanzados y exportación a PDF/Excel.
